# Interval Timer

A small, dependency-free PWA interval timer. Configure a work duration, a
pause duration, and a number of repetitions; the timer runs
`(work + pause) × repetitions`. One beep marks the end of each work interval,
two beeps mark the end of each pause, and three beeps mark the end of the
final cycle. Works on desktop and mobile, installs to the home screen, and
runs offline.

## Features

- Work / pause / repetitions configuration, persisted in `localStorage`
- Timestamp-based countdown (stays accurate even when timers are throttled)
- Audio cues via the Web Audio API (1 beep after work, 2 after pause,
  3 when the workout completes), plus vibration on devices that support it
- Screen wake lock while a workout is running
- Offline-capable via a service worker; installable via the web app manifest

## Run locally

```sh
./serve.sh          # serves on 0.0.0.0:8123 and prints the LAN URL
./serve.sh 9000     # custom port
```

Open the printed `http://<your-ip>:8123` URL on a phone on the same Wi-Fi to
test on mobile. Any other static file server works too.

## Deploy

The app is deployed via GitHub Pages from `master` at
<https://fuchstim.github.io/interval-timer/>. Pushes to `master` go live
automatically.

It can also be copied to any other static host (Netlify, etc.). The app must
be served over **HTTPS** (or localhost) for the service worker, install
prompt, and wake lock to be available. All paths are relative, so hosting
under a subpath works.

## Notes

- Browsers only allow sound after a user gesture; the audio context is
  unlocked when you tap **Start**.
- While a workout runs, the app loops a silent audio track (`silence.wav`).
  This puts the iOS audio session into "playback" mode so beeps are audible
  even when the iPhone ringer switch is on silent. Side effect: starting the
  timer may pause music playing in another app.
- On mobile, the wake lock keeps the screen on so beeps fire on time. The
  wake lock (like the service worker) requires HTTPS — over plain-HTTP LAN
  testing via `serve.sh`, keep the screen awake yourself or raise the
  auto-lock timeout. If the screen is locked manually, browsers freeze page
  timers and audio until the app is foregrounded again.
- Icons are generated by `tools/make_icons.py` (Python stdlib only).
